## Project Overview
This project predicts diabetes in expecting mothers using Machine Learning Algorithms and Hyperparameter Tuning.

## Algorithms Used
- Decision Tree
- K-Nearest Neighbour (KNN)
- Random Forest
- Naive Bayes
- Support Vector Machine (SVM)

## Best Model
Support Vector Machine (SVM)

## Performance Metrics
| Metric | Value |
|---|---|
| Accuracy | 88% |
| Precision | 86% |
| Recall | 76% |
| F1-Score | 80% |

Pima Indians Diabetes Dataset
